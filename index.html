Ping Pong!

Rules of the Game

  • Input any number between 1 and 200
  • If your number is divisible by 3, the output is all the numbers before it and itslef is dipsplayed as "Ping".
  • If your number is divisible by 5, the program outputs all the numbers before it with itself being output as "Pong".
  • If your number is divisible by 15, the output is all other numbers before it, and the number is displayed as "PingPong".
